BS

A minimal embeddable scripting language with sane defaults and strong typing 

fn factorial(n) {
    if n < 2 {
        return 1;
    }

    return n * factorial(n - 1);
}

io.println("5! = \(factorial(5))");

$ bs factorial.bs
5! = 120
